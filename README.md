
# UC11
  O propósito deste realatório é exibir todos os comandos utilizados na inserção do versionamento no repositório local e remoto. 
  O propósito deste relatório é exibir todos os comandos utilizados na criação do versionamento no repositório local e remoto, pra este versionamento foram 
utilizados o Git Bash e github.

 1- Primeiro comando utilizado foi git init para dar inicio ao versionamento local. 
 2- O comando git status está sendo utilizado para verificar e confirmar se o arquivo foi inserido no repositório local.
 3- Utilizei o comando git commit para inserir a descrição do versionamento.
 4- Repositório remoto no GITHUB foi criado para armazenar o projeto LeiloesTDSat.
 5- git remote utilizado pra criar um versionamento remoto no github.
 6- Utilizei o comando git pull origin master para realizar a sicronização com o repositório local.
